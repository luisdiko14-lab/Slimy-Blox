## Packages
framer-motion | Essential for smooth UI animations and game feel
clsx | Utility for conditional class names
tailwind-merge | Utility for merging tailwind classes

## Notes
The game uses a custom 2D engine built with React state and requestAnimationFrame.
Rank is stored in localStorage.
Backend logging via POST /api/logs.
Font choice: 'VT323' and 'Press Start 2P' for that retro terminal feel.
